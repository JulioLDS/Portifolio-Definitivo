# Portifolio-Definitivo

Portfólio profissional focado em apresentar projetos, habilidades e experiências de forma moderna, acessível e responsiva.

## 🚀 Visão Geral

Este projeto é um site de apresentação pessoal / portfólio, criado com:
- HTML
- CSS (com design glassmorphism)
- JavaScript

O layout foi pensado para funcionar bem em diversos dispositivos (desktop, tablet e mobile).

## 🧩 Estrutura do Projeto

- `index.html` - página principal
- `termos.html` - página de termos de uso
- `privacidade.html` - página de política de privacidade
- `cookies.html` - página de cookies
- `css/style.css` - estilos do site
- `js/main.js` - scripts de interação
- `data/` - dados de projetos, certificados e tecnologias em JSON
  - `projects.json`
  - `certificates.json`
  - `technologies.json`
- `assets/images/` - imagens usadas no portfólio

## 🎯 Funcionalidades

- Navegação responsiva e intuitiva
- Área de apresentação de projetos
- Lista de tecnologias e certificados
- Suporte a múltiplas páginas informativas (termos, privacidade, cookies)
- Visual agradável com efeito glassmorphism

## 🛠️ Como executar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/juliolds/Portifolio-Definitivo.git
   ```
2. Abra `index.html` no navegador.

> Dica: use o Live Server do VS Code para recarregar rápido durante desenvolvimento.

## 📌 Deploy

O site está hospedado via GitHub Pages e pode ser acessado diretamente.
